Console-Based Text Editor (C++)

This console-based text editor was the first time I was exposed to Data Structures. In this project, users could open a text file as a command-line argument and could interact with the file. This project was completed with each line being a node in a Linked List, and I wanted to display this as I feel that a linked list is a fundamental data structure.
